# Market and Shop — Frontend

Vite + React SPA for the Market and Shop vendor marketplace.

## Setup

```bash
cp .env.example .env.local
npm install
npm run dev
```

Configure `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` from your Supabase project dashboard.

## Deploy

Deploy the `frontend/` directory to Vercel with Root Directory set to `frontend`. See the root `README.md` for full instructions.
